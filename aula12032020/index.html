TAGS FUNÇÃO
ol serve para montar uma lista ordenada
ul serve para montar uma lista não ordenada
p faz paragráfo
b deixa em negrito quando utilizada na versão 4 do html
br quebra linha
i deixa a letra em itálico
strong deixa a letra em negrito quando utilizada na versão 5 do html
hr faz uma linha
tr linhas de uma tabela
td colunas de uma tabela
!-- cria um comentário
li um item da lista
ATRIBUTOS FUNÇÃO
id facilita a identificação de elementos que vão receber um estilo CSS ou até mesmo uma função Javascript
style usada para estilizar todo o documento (ex:colorir a página)
class identifica um grupo de elementos. Através dela, pode-se atribuir formatação a vários elementos de uma vez
